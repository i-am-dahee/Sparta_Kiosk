### ☕ Console Kiosk Program (JAVA)
> 자바로 만든 콘솔 기반 키오스크 프로그램입니다.
> 
> 메인 메뉴 → 서브 메뉴 → 장바구니 → 할인 → 주문 플로우로 돌아갑니다.

---

### ✅ 주요 기능
- 카테고리 선택 : Burgers, Drinks, Desserts 등 카테고리 선택
- 메뉴 선택 : 원하는 메뉴를 선택해 장바구니 담기
- 장바구니 관리 : 장바구니 조회/추가/삭제
- 할인 적용 : 사용자 타입에 맞는 할인율 적용
- 주문 여부 확인 : 주문 확정 또는 취소
- 종료 : 종료 메뉴 (0)을 통해 프로그램 종료

---

### 📦 프로젝트 구조
```
src/
└─ kiosk01/
   ├─ Main.java          // 프로그램 실행 진입점 (main 메서드)
   ├─ Kiosk.java         // 전체 프로그램 플로우 제어
   ├─ Menu.java          // 카테고리 정보 (예: Burgers, Drinks 등)
   ├─ MenuItem.java      // 개별 메뉴 아이템 정보 (이름, 가격, 설명)
   ├─ Cart.java          // 장바구니 기능 (추가, 삭제, 총합 계산)
   ├─ CartItem.java      // 장바구니 내 개별 항목 클래스 (메뉴 + 수량)
   ├─ Order.java         // 주문 정보 저장 (총 금액, 할인 적용 등)
   ├─ OrderStatus.java   // 주문 상태(enum) - PENDING, PAID
   ├─ Discount.java      // 할인 정책 관련 클래스 (유형별 할인율 계산)
   ├─ Usertype.java      // 사용자 유형(enum) - 국가유공자, 군인, 학생, 일반
   └─ InputHelper.java   // 사용자 입력 처리 유틸 (예외 처리 및 입력 검증)
```
